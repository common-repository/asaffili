asAffili is an affiliate tool. It allows you to import csv data feeds provided by affiliate networks.

== Description ==

= asAffili is an simply and easy-to-use Plugin to create a product-site with products from affiliate-datafeeds =

You can:

* Create data feeds imports
* Link product-categories to own categories
* Display the imported products

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/asaffili` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the asAffili->Options screen to configure the plugin
4. Create a Page, that display the imported products with Shortcode
5. Create the imports to import the data-feeds from affiliate-networks
